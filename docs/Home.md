# Welcome to the gcli Wiki

`gcli` is a powerful Go-based command-line interface for interacting with the Grafana API.

## Project Goals

The main goal of `gcli` is to provide a fast, efficient, and interactive way to manage Grafana resources without leaving your terminal. Whether you are managing multiple organizations, configuring data sources, or deploying dashboards, `gcli` aims to simplify your workflow.
